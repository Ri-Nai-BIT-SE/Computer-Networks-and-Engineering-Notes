# 2.3 物理层设备

本节介绍工作在物理层（第一层）的网络设备，它们的主要功能是处理和转发比特流。

## 核心概念

### 1. 中继器 (Repeater)

*   **定义**: 一种信号再生设备，用于连接**相同类型**的局域网段。

*   **功能**:
    *   对衰减和失真的信号进行**整形和放大**。
    *   再生出与原始信号相同的比特流。
    *   主要目的是延长网络的传输距离。

*   **特点**:
    *   工作在物理层，只处理比特流。
    *   不理解帧、MAC地址或IP地址等高层概念。
    *   两端网络必须相同（如都是10BASE-T以太网）。
    *   连接的两个网段在逻辑上是一个网络（同一个广播域和冲突域）。

*   **限制**:
    *   **5-4-3规则**: 在10BASE-T以太网中:
        *   最多使用4个中继器。
        *   最多连接5个网段。
        *   其中只有3个网段可以挂接主机。
        *   目的是保证信号传播时延在可控范围内。

### 2. 集线器 (Hub)

*   **定义**: 一个**多端口的中继器**。

*   **功能**:
    *   当一个端口收到数据时，将数据**广播**到**所有其他端口**。
    *   不进行任何帧的处理或过滤。
    *   纯粹的物理层设备。

*   **工作模式**:
    *   **半双工**: 所有设备共享同一个带宽。
    *   所有端口都在同一个**冲突域**内。
    *   同一时刻只能有一个设备发送数据。

*   **分类**:
    *   **无源集线器**: 
        *   仅做信号连接。
        *   不放大或再生信号。
        *   很少使用。
    *   **有源集线器**:
        *   对信号进行再生和放大。
        *   是最常见的类型。
    *   **智能集线器**:
        *   具有网络管理功能。
        *   可通过网管系统进行监控。

*   **缺点**:
    *   共享带宽，性能随连接设备数增多而急剧下降。
    *   所有设备在同一个冲突域内，冲突概率高。
    *   不安全，所有设备都能监听到所有通信。
    *   现已基本被**交换机**取代。

### 3. 中继器与集线器的比较

*   **相同点**:
    *   都工作在物理层。
    *   只处理比特流，不理解高层协议。
    *   都用于扩展网络距离。
    *   都保持了同一个冲突域。

*   **不同点**:
    *   中继器通常是两个端口，集线器有多个端口。
    *   中继器主要用于延长距离，集线器主要用于连接多台设备。

## 易考点 & 难点

### 易考点:
*   中继器的功能（信号再生）和用途（延长距离）。
*   集线器的本质（多端口中继器）和工作方式（广播）。
*   5-4-3规则的具体含义。
*   集线器的主要缺点。

### 难点:
*   **冲突域的概念**:
    *   理解为什么集线器的所有端口都在一个冲突域内。
    *   与交换机（每个端口一个冲突域）的区别。

*   **物理层设备的局限性**:
    *   为什么不能识别MAC地址和IP地址。
    *   为什么被更高层设备（如交换机）替代。

*   **5-4-3规则的原理**:
    *   信号在传输过程中的衰减和时延问题。
    *   为什么需要这样的限制规则。
