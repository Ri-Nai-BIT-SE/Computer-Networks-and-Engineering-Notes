# 2.2.1 数据通信系统的模型

## 数据通信系统的组成

### 源系统（发送端）
- **源点（Source）**：产生要传输的数据
- **发送器（Transmitter）**：将数字比特流转换为适合在传输媒体上传输的信号

### 传输系统（传输网络）
- **传输媒体**：物理信道
- **可能包含**：交换节点、中继器等

### 目的系统（接收端）
- **接收器（Receiver）**：将信号转换为数字比特流
- **终点（Destination）**：接收数据

## 信号转换

### 调制（Modulation）
- **定义**：将数字信号转换为模拟信号的过程
- **作用**：使数字信号能够在模拟信道上传输
- **实现**：调制解调器（Modem）的发送功能

### 解调（Demodulation）
- **定义**：将模拟信号转换为数字信号的过程
- **作用**：从模拟信号中恢复原始数字数据
- **实现**：调制解调器（Modem）的接收功能

## 数据通信过程
1. 源点产生数据
2. 发送器将数据转换为信号（调制）
3. 信号通过传输系统传输
4. 接收器将信号转换为数据（解调）
5. 终点接收数据
