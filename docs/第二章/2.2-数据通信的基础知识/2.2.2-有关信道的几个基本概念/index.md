# 2.2.2 有关信道的几个基本概念

## 通信方式

### 单工通信
- **定义**：只有一个方向的通信，没有反方向的交互
- **特点**：单向传输
- **例子**：无线电广播、电视广播

### 半双工通信
- **定义**：通信的双方都可以发送信息，但不能同时发送（也不能同时接收）
- **特点**：双向但不同时
- **例子**：对讲机

### 全双工通信
- **定义**：通信的双方可以同时发送和接收信息
- **特点**：双向同时
- **例子**：电话、计算机网络

## 基带信号与调制

### 基带信号
- **定义**：来自信源的信号，往往包含有较多的低频成分，甚至有直流成分
- **特点**：不适合在信道中直接传输

### 调制
- **目的**：将基带信号转换为适合在信道中传输的信号
- **两种方式**：
  1. **编码**：基带调制，仅对基带信号的波形进行变换
  2. **带通调制**：使用载波进行调制，把基带信号的频率范围搬移到较高的频段

## 常用编码方式

### 不归零制（NRZ）
- **特点**：正电平代表1，负电平代表0
- **缺点**：
  - 难以确定一位的结束和另一位的开始
  - **没有自同步能力**：连续发一串1时，时钟容易跑偏，接收端无法从信号中提取时钟频率

### 归零制（RZ）
- **特点**：正脉冲代表1，负脉冲代表0
- **优点**：每个码元中间都有电平跳变

### 曼彻斯特编码
- **特点**：
  - 位周期中心的向上跳变代表0
  - 位周期中心的向下跳变代表1
  - 位周期中心的**跳变**既代表信息，也提供时钟信号（**具有自同步能力**）
- **优点**：具有自同步能力
- **缺点**：频带宽度是基带信号的2倍
- **应用**：以太网

### 差分曼彻斯特编码
- **特点**：
  - 在每一位的中心处始终都有跳变
  - 利用**位开始边界**有无跳变来表示：
    - 位开始边界**有**跳变 → 0
    - 位开始边界**无**跳变 → 1
- **优点**：
  - 具有自同步能力
  - 抗干扰能力更强
- **应用**：令牌环网

## 带通调制方式

### 调幅（AM - Amplitude Modulation）
- **原理**：改变载波的振幅
- **特点**：简单但抗干扰能力差

### 调频（FM - Frequency Modulation）
- **原理**：改变载波的频率
- **特点**：抗干扰能力较强

### 调相（PM - Phase Modulation）
- **原理**：改变载波的相位
- **特点**：抗干扰能力强

### 正交振幅调制（QAM）
- **原理**：同时改变载波的振幅和相位
- **特点**：可以传输更多的信息

## 考试重点：编码波形画图题

### 曼彻斯特编码（Manchester）波形画法

#### 谢希仁教材定义（做题前看清题目要求）
- **0**：位周期中心从**低变高**（$\nearrow$，上升沿）
- **1**：位周期中心从**高变低**（$\searrow$，下降沿）

#### 画图技巧
1. 每个位周期的**中间**必须画一笔跳变（这是自同步的关键）
2. 跳变方向根据数据位决定：
   - 数据为0：中间向上跳变
   - 数据为1：中间向下跳变
3. 如果前一位结尾是低电平，下一位是0，则中间从低跳到高
4. 如果前一位结尾是高电平，下一位是1，则中间从高跳到低

#### 示例：画出 101101 的曼彻斯特编码波形
```
数据：  1    0    1    1    0    1
波形：  ┌─┐  ┌─┐  ┌─┐  ┌─┐  ┌─┐  ┌─┐
       │ └──┘ └──┘ └──┘ └──┘ └──┘ │
       └────────────────────────────┘
       中心跳变：↓    ↑    ↓    ↓    ↑    ↓
```

### 差分曼彻斯特编码（Differential Manchester）波形画法

#### 核心规则
- **中间永远有跳变**（用于同步）
- **位开始边界**的跳变与否表示数据：
  - 遇到数字 **0**：在周期起始处**有跳变**（和上一位结尾电平反着来）
  - 遇到数字 **1**：在周期起始处**无跳变**（接着上一位结尾电平画）

#### 记忆口诀
**"0变1不变"**（指起始边界的跳变）

#### 画图步骤
1. 确定初始电平（通常从高电平或低电平开始）
2. 对于每一位：
   - 如果是**0**：在起始边界**跳变**（改变电平），然后在中间再跳变一次
   - 如果是**1**：在起始边界**不跳变**（保持上一位结尾的电平），然后在中间跳变一次
3. 中间始终有跳变，这是同步信号

#### 示例：画出 101101 的差分曼彻斯特编码波形
```
数据：  1    0    1    1    0    1
波形：  ┌─┐  ┌─┐  ┌─┐  ┌─┐  ┌─┐  ┌─┐
       │ └──┘ └──┘ └──┘ └──┘ └──┘ │
       └────────────────────────────┘
       边界：无跳变 有跳变 无跳变 无跳变 有跳变 无跳变
       中间：始终有跳变
```

#### 注意事项
- 不同教材对曼彻斯特编码的定义可能相反，做题前一定要看清题目要求
- 差分曼彻斯特编码的抗干扰能力更强，因为它是通过边界跳变来判断的
