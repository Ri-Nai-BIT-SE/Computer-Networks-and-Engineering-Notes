# 3.4.1 在物理层扩展以太网

## 扩展方法

### 使用光纤
- **目的**：扩大地理覆盖范围
- **方法**：使用光纤和一对光纤调制解调器连接两个集线器
- **优点**：
  - 可以连接距离较远的站点
  - 光纤传输距离远，抗干扰能力强

### 使用集线器扩展
- **方法**：使用多个集线器连接更多的站点
- **特点**：
  - 所有站点仍在同一个碰撞域内
  - 逻辑上仍是总线网
  - 不能隔离碰撞域

## 碰撞域

### 定义
- **碰撞域**：在同一个碰撞域内，任意两个站点同时发送数据都会产生碰撞
- **特点**：使用集线器扩展的所有站点都在同一个碰撞域内

### 问题
- 碰撞域越大，碰撞概率越高
- 网络性能会随着站点数量增加而下降

## 物理层扩展的局限性

1. **不能隔离碰撞域**：所有站点共享同一个碰撞域
2. **不能提高总吞吐量**：总吞吐量受限于单个站点的发送速率
3. **广播风暴**：广播帧会在整个网络中传播

## 改进方案
- 使用**交换机**在数据链路层扩展以太网，可以隔离碰撞域
