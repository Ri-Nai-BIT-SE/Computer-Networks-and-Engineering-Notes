# 3.3.4 以太网的信道利用率

## 参数 $a$ 的定义 ⭐ **选择题考点**

### 公式
$$a = \frac{\tau}{T_0}$$

其中：
- $\tau$：端到端的传播时延
- $T_0$：发送一个帧所需的时间

### 物理意义
- $a$ 表示传播时延与发送时延的比值
- **$a$ 越小，信道利用率越高** ⭐
- **结论**：$a$ 应尽可能小（$a \to 0$），表示一发生碰撞就能被检测出来，信道利用率高

### 信道利用率
- **理想情况**：$a \to 0$，即传播时延远小于发送时延
- **实际情况**：$a$ 越大，信道利用率越低

## 信道利用率的计算

### 最大信道利用率
$$S_{max} = \frac{T_0}{T_0 + \tau} = \frac{1}{1 + a}$$

### 说明
- 当 $a \to 0$ 时，$S_{max} \to 1$（理想情况）
- 当 $a$ 增大时，$S_{max}$ 减小

## 影响信道利用率的因素

1. **帧长度**：帧越长，$T_0$ 越大，$a$ 越小，利用率越高
2. **传播时延**：传播时延越大，$\tau$ 越大，$a$ 越大，利用率越低
3. **碰撞**：碰撞会降低信道利用率

## 提高信道利用率的方法

1. **增加帧长度**：但受MTU限制
2. **减少传播时延**：使用更短的链路或更快的传输媒体
3. **减少碰撞**：使用交换机代替集线器
