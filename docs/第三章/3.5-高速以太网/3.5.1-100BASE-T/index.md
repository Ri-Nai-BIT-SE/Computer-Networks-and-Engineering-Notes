# 3.5.1 100BASE-T（快速以太网）

## 100BASE-T概述

### 定义
- **100BASE-T**：快速以太网（Fast Ethernet）
- **速率**：100Mbps
- **标准**：IEEE 802.3u

### 命名规则
- **100**：传输速率为100Mbps
- **BASE**：基带传输
- **T**：双绞线（Twisted pair）

## 100BASE-T的变体

### 100BASE-TX
- **传输媒体**：2对5类UTP或STP
- **距离**：100米
- **编码**：4B/5B编码

### 100BASE-T4
- **传输媒体**：4对3类UTP
- **距离**：100米
- **编码**：8B/6T编码

### 100BASE-FX
- **传输媒体**：2根光纤
- **距离**：可达2公里
- **编码**：4B/5B编码

## 特点

### 与10BASE-T的兼容性
- 使用相同的帧格式
- 使用相同的CSMA/CD协议
- 可以使用相同的集线器或交换机（如果支持）

### 自动协商
- **功能**：自动检测对方的工作模式（10Mbps或100Mbps）
- **标准**：IEEE 802.3u的自动协商功能
- **工作方式**：通过发送快速链路脉冲（FLP）进行协商

## 应用
- 快速以太网广泛应用于局域网
- 是10Mbps以太网的升级版本
