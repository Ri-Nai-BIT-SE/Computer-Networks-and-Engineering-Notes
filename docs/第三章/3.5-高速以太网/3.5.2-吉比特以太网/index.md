# 3.5.2 吉比特以太网

## 吉比特以太网概述

### 定义
- **速率**：1Gbps（1000Mbps）
- **标准**：IEEE 802.3z（光纤和屏蔽双绞线）和IEEE 802.3ab（非屏蔽双绞线）

### 命名规则
- **1000**：传输速率为1000Mbps
- **BASE**：基带传输
- **X/T**：传输媒体类型

## 吉比特以太网的变体

### 1000BASE-X
- **1000BASE-SX**：短波长光纤，550米
- **1000BASE-LX**：长波长光纤，5公里
- **1000BASE-CX**：屏蔽双绞线，25米

### 1000BASE-T
- **传输媒体**：4对5类UTP
- **距离**：100米
- **编码**：4D-PAM5编码

## 载波延伸（Carrier Extension）

### 问题
- 吉比特以太网的最小帧长仍为64字节
- 但争用期变为512字节（0.512μs）
- 如果帧长小于512字节，需要填充到512字节

### 解决方案
- **载波延伸**：在帧的后面添加载波延伸字段
- **目的**：确保帧的长度至少为512字节
- **特点**：载波延伸字段不包含数据，只是用于延伸载波信号

## 分组突发（Packet Bursting）

### 问题
- 如果有很多短帧，载波延伸会降低效率

### 解决方案
- **分组突发**：允许发送端连续发送多个短帧
- **工作原理**：
  1. 第一个帧使用载波延伸
  2. 后续的帧可以连续发送，不需要载波延伸
  3. 帧之间只需要很小的间隔（96比特时间）

### 优点
- 提高了短帧的传输效率
- 减少了载波延伸的开销

## 全双工方式
- 吉比特以太网支持全双工通信
- 全双工方式下不需要CSMA/CD协议
- 不存在碰撞问题

## 应用
- 吉比特以太网广泛应用于高速局域网
- 常用于服务器连接、骨干网络等场景
