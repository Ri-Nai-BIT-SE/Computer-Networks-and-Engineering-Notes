# 3.1.1 数据链路和帧

## 链路与数据链路的区别

### 链路（Link）
- **定义**：从一个节点到相邻节点的一段物理线路（有线或无线）
- **特点**：只是一条路径的组成部分

### 数据链路（Data Link）
- **定义**：除了物理线路外，还必须有通信协议来控制这些数据的传输
- **组成**：物理链路 + 实现协议的硬件和软件
- **特点**：在数据链路层，数据以帧（Frame）为单位传输

## 适配器（网卡）
- **定义**：网络适配器（Network Adapter），也称为网络接口卡（NIC）
- **功能**：
  1. **串/并行转换**：将计算机内部的并行数据转换为串行数据
  2. **缓存**：临时存储数据
  3. **实现以太网协议**：在硬件和软件中实现数据链路层和物理层的功能

## 帧（Frame）
- **定义**：数据链路层的数据传输单位
- **组成**：帧首部 + 数据部分 + 帧尾部
- **作用**：在数据链路层，数据以帧为单位进行传输
