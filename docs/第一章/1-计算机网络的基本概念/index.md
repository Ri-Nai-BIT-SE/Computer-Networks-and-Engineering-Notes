# 1.1 计算机网络的基本概念

本节是计算机网络的基础入门，主要定义了什么是计算机网络，其与传统计算机系统的区别，以及网络在当今社会中的重要作用。

### 核心概念

#### 1. 计算机网络的重要性

*   **时代特征**：21世纪是数字化、网络化和智能化的时代，网络已成为信息社会的命脉。
*   **新兴技术**：人工智能、云计算、物联网、大数据、移动互联等都与网络密不可分。
*   **因特网发展**：
    *   从最初的教育科研网络发展成为商业网络。
    *   **网络**：将多台计算机连接在一起。
    *   **因特网 (Internet)**：将多个网络连接在一起，是人类通信方面最大的变革。

#### 2. 计算机网络的定义

*   **官方定义**：由**自主计算机**通过**互联**技术组成的**集合体**。
    *   **自主计算机 (Autonomous Computer)**：
        *   指具备独立硬件和软件，能完整实现计算功能的计算机。
        *   在网络中，每台计算机都拥有独立的操作系统，不从属于任何一台主机。
    *   **互联 (Interconnected)**：
        *   指计算机之间能够实现相互通信。
        *   通过通信线路及互联设备（如交换机、路由器）连接。
    *   **集合体 (Collection)**：
        *   所有通过通信线路和互联设备连接起来的自主计算机的整体。

#### 3. 计算机网络与主从式计算机系统的区别

*   **主从式系统**：由一台主机控制多台从属终端或前端处理机。特点是资源集中，终端依赖主机。
*   **计算机网络**：每台计算机都是独立的，拥有自己的操作系统，并通过网络共享资源，无主从关系。

#### 4. 计算机网络与分布式计算机系统的区别

*   **分布式系统**：
    *   “存在一个能为用户自动管理资源的**网络操作系统**。”
    *   整个网络对用户是**透明的**，用户感觉像在使用一台大型计算机。
    *   **关系**：计算机网络为分布式系统提供了技术基础，分布式系统是计算机网络发展的高级阶段。
*   **计算机网络**：提供互联互通的基础设施，不一定包含统一的网络操作系统来自动管理资源。用户通常能感知到网络的存在。

### 易考点 & 难点

*   **易考点**：计算机网络的定义（自主计算机、互联、集合体），以及其与主从式、分布式系统的本质区别。尤其要注意“自主性”和“透明性”的对比。
*   **难点**：区分分布式系统中的“透明性”与计算机网络中“资源共享”的含义，理解分布式系统是网络更高级的应用。

### 计算机网络的组成

*   **硬件**：
    *   **计算机（主机/端系统）**：个人计算机、大型计算机、客户机、工作站、服务器。
    *   **通信设备（中间系统）**：交换机、路由器（用于转发数据）。
    *   **接口设备**：网卡（NIC）、调制解调器（Modem）。
    *   **传输介质**：双绞线、同轴电缆、光纤、无线链路。
*   **软件**：
    *   **通信协议**：数据传输规则，如TCP/IP。
    *   **应用软件**：如WWW、EMAIL、FTP等。

### 计算机网络的功能

*   **共享资源访问**：
    *   万维网 (WWW) 访问、远程登录、网络文件访问。
*   **远程用户通信**：
    *   电子邮件 (Email)、IP电话、网络会议。
*   **网上事务处理**：
    *   电子商务、电子政务、电子金融、远程教育、远程医疗。
