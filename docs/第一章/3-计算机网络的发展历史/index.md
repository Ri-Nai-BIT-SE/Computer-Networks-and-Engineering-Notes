# 1.3 计算机网络的发展历史

本节回顾了计算机网络从军事需求到商业普及，再到全球互联的演进过程，重点介绍了ARPANET和因特网的发展阶段。

### 核心概念

#### 1. 计算机网络的产生背景

*   **冷战产物**：20世纪60年代美苏冷战时期，美国国防部ARPA（Advanced Research Project Agency）提出研制**生存性 (survivability)** 强的网络。
*   **传统电信网缺点**：电路交换的电信网在链路中断时整个通信会中断，需要重新建立连接，延误时间。
*   **新型网络需求**：
    *   用于计算机之间的数据传送（而非打电话）。
    *   连接不同类型的计算机。
    *   所有网络节点同等重要，提高生存性。
    *   通信时需有冗余路由。
    *   网络结构尽可能简单，同时可靠传送数据。

#### 2. ARPANET的成功

*   **根本变化**：将早期"以主机为中心"的星形网（终端共享昂贵中心主机）转变为"以网络为中心"的分组交换网（主机在外围，用户通过网络共享多种资源）。

#### 3. 因特网（Internet）的发展阶段

因特网基础结构演进的三个阶段并非截然分开，而是逐步重叠：

*   **第一阶段 (ARPANET时期)**：
    *   ARPANET是第一个分组交换网。
    *   ARPA研究网络互连技术。
    *   **1983年**：TCP/IP协议成为标准协议。
    *   ARPANET分解为：
        *   **ARPANET**：科研网。
        *   **MILNET**：军用计算机网络。
    *   **1983-1984年**：形成因特网 (Internet)。
    *   1990年：ARPANET正式关闭。

*   **第二阶段 (NSFNET时期)**：
    *   **1986年**：美国国家科学基金会 (NSF) 建立国家科学基金网 (NSFNET)。
    *   **NSFNET三级结构**：主干网 → 地区网 → 校园网（或企业网）。各网络之间通过路由器连接。
    *   **1991年**：美国政府将因特网主干网转交给私人公司经营，开始收费。
    *   1993年：主干网速率提高到 $45 \text{ Mb/s (T3速率)}$。

*   **第三阶段 (ISP多层次结构时期)**：
    *   **1993年至今**：NSFNET逐渐被商用**因特网服务提供商 (ISP - Internet Service Provider)** 网络代替。
    *   **1994年**：创建4个**网络接入点 (NAP - Network Access Point)**，用于交换因特网流量。
    *   **特点**：形成**多层次ISP结构**的因特网。用户通过本地ISP接入，本地ISP连接到第二层ISP，再连接到第一层ISP或NAP，实现全球互联。

#### 4. `internet` 与 `Internet`

*   **`internet` (普通名词)**：泛指一般的互连网（互联网）。
*   **`Internet` (专有名词)**：特指世界范围的互连网（因特网），使用TCP/IP协议族，前身是ARPANET。

#### 5. 计算机网络在我国的发展

*   **1980年代**：我国开始计算机联网实验，建成第一个公用分组交换网CNPAC，公安、银行、军队等部门建立专用广域网，大量局域网普及。
*   **1994年4月20日**：我国正式连入因特网。
*   **1994年5月**：中科院高能物理所设立我国第一个万维网服务器。
*   **1994年9月**：中国公用计算机互联网CHINANET正式启动。
*   **重要网络**：中国教育和科研计算机网CERNET。

### 易考点 & 难点

*   **易考点**：
    *   ARPANET产生的背景和目标，以及它对网络发展的关键影响。
    *   因特网发展的三个阶段（ARPANET、NSFNET、ISP多层次结构）及其主要特征。
    *   `internet` 与 `Internet` 的区别。
*   **难点**：理解因特网从科研网络到商业网络的转变，以及多层次ISP结构如何实现全球网络的互联和流量交换。
