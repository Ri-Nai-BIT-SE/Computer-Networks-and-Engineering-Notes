# 1.4 计算机网络的标准化工作及相关组织

本节介绍了计算机网络领域主要的标准化组织，以及因特网标准制定过程，强调了标准对网络互操作性的重要性。

### 核心概念

#### 1. 国际标准化组织 (ISO)

*   **性质**：自发不缔约组织。
*   **职责**：在TC7下成立SC16（后改为SC21），以"开放系统互连"（OSI）为目标，研究和制定相关标准，特别是七层模型中的高四层及整个参考模型。
*   **相关委员会**：SC6负责低三层及数据通信相关标准。
*   **重要性**：为OSI参考模型奠定了基础。

#### 2. 国际电信联盟 (ITU)

*   **前身**：CCITT（国际电报电话咨询委员会）。
*   **现称**：国际电信联盟电信标准化局 (ITU-T)。
*   **职责**：负责有关通信标准的研究和制定，尤其在电信领域具有权威性。

#### 3. 美国国家标准学会 (ANSI)

*   **性质**：由制造商、用户通信公司组成的非政府组织。
*   **角色**：美国指定的ISO投票成员，美国的自发标准情报交换机构。
*   **下属机构**：
    *   **电子工业协会 (EIA)**：主要涉及OSI物理层标准的制定。
    *   **电气和电子工程师学会 (IEEE)**：主要研究最低两层（物理层和数据链路层）及局域网的相关标准（如IEEE 802系列）。
