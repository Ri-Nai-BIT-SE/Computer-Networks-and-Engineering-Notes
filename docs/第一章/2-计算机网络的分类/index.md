# 1.2 计算机网络的分类

本节主要介绍计算机网络的多种分类方法，并通过传输技术和距离尺度两个重要因素深入探讨局域网（LAN）、城域网（MAN）和广域网（WAN）的特点。

### 核心概念

#### 1. 计算机网络的分类方法

网络可以从不同维度进行分类：

*   **作用范围**：
    *   **个人区域网 (PAN)**：Personal Area Network，1米见方，如蓝牙。
    *   **局域网 (LAN)**：Local Area Network，同一房间/建筑物/校园 (10m-1km)。
    *   **城域网 (MAN)**：Metropolitan Area Network，同一城市 (10km)。
    *   **广域网 (WAN)**：Wide Area Network，同一国家/洲 (100km-10000km)。
*   **拓扑结构**：总线型、环形、星型、树型、网状型、混合型。
*   **通信协议**：TCP/IP、ATM、X.25、FDDI。
*   **交换功能**：电路交换网、报文交换网、分组交换网。
*   **传输媒体**：双绞线网、光纤网、卫星网、无线网。
*   **服务对象**：企业网、校园网、园区网、政府网。
*   **使用者**：专用网、公用网。
*   **应用性质**：远程教育网、证券网、税务网。

#### 2. 分类方法中的重要因素

**a. 传输技术**

*   **广播式 (Broadcasting)**：
    *   **特点**：网络中只有一个共享的通信信道。
    *   **通信过程**：任何机器发送的分组（packet）都会被所有其他机器接收。分组中包含目标地址，接收机器会检查地址域，匹配则接收，否则忽略。
    *   **通信种类**：
        *   **广播**：发送给网络中所有机器。
        *   **多播 (Multicasting)**：发送给网络中特定组的机器。
    *   **适用场景**：通常用于地理位置较小、局部化的网络（如局域网）。

*   **点到点式 (Point-to-point)**：
    *   **特点**：网络由多个连接构成，每个连接对应一对机器。
    *   **通信过程**：分组从源端到目的地可能需要经过一个或多个中间机器（路由器）。
    *   **适用场景**：通常用于大型网络（如广域网）。

**b. 距离尺度**

| 传输器之间的距离 | 处理器所在的位置 | 例子          |
| :--------------- | :--------------- | :------------ |
| 1 m              | 一米见方的范围   | 个人区域网    |
| 10 m             | 同一房间         | 局域网 (LAN)  |
| 100 m            | 同一建筑物内     |               |
| 1 km             | 同一校园         |               |
| 10 km            | 同一城市         | 城域网 (MAN)  |
| 100 km           | 同一国家         | 广域网 (WAN)  |
| 1000 km          | 同一个洲         |               |
| 10000 km         | 同一个行星       | 因特网 (Internet) |

#### 3. 局域网 (LAN)

*   **特点**：
    *   覆盖有限地理范围（公司、机关、校园、工厂）。
    *   高数据传输速率（$10Mbps \sim 100Gbps$），低误码率。
    *   通常属于一个单位所有，易于建立、维护、扩展。
    *   介质访问控制方法分为共享介质式和交换式。
*   **拓扑结构示例**：星型（通过Hub/Switch连接），总线型。

#### 4. 城域网 (MAN)

*   **特点**：
    *   覆盖一个城市。
    *   介于广域网与局域网之间的高速网络。
    *   IEEE 802.6（DBDQ）是其标准之一。
    *   现代以太网技术（千兆、万兆以太网）也扩展到MAN领域。
    *   可使用SONET/SDH、WDM、ATM、FDDI等技术。
*   **典型例子**：有线电视网。

#### 5. 广域网 (WAN)

*   **组成**：
    *   **资源子网**：连接到网络上的主机。
    *   **通信子网**：由交换节点（路由器）和传输线路组成。
*   **技术特点**：
    *   覆盖范围从几十公里到几千公里（国家、地区、洲际）。
    *   通信子网主要使用**分组交换技术**。
    *   目标是实现不同地区计算机系统的互连，达到资源共享。

### 易考点 & 难点

*   **易考点**：
    *   LAN、MAN、WAN 的概念、覆盖范围和主要特点对比。
    *   广播式和点到点式传输技术的原理和适用场景。
    *   广域网由资源子网和通信子网组成。
*   **难点**：理解各种网络类型背后的传输技术（共享信道与点到点路径）如何影响网络的规模和性能。
