# 1.3.1 互联网的边缘部分

## 组成
- **主机（Host）**：连接在互联网上的所有主机，又称**端系统（End System）**
- **包括**：个人计算机、服务器、移动设备等
- **作用**：由用户直接使用，进行通信（传送数据、音频、视频）和资源共享

## 端系统之间的通信方式
- **"计算机之间通信"**实际上是指"主机A的某个**进程**和主机B上的另一个**进程**进行通信"

### 客户-服务器方式（C/S方式）
- **Client（客户）**：服务请求方（主动发起）
- **Server（服务器）**：服务提供方（被动等待）
- **特点**：
  - 客户程序需知道服务器地址
  - 服务器需强大的硬件和高级操作系统支持
  - 服务器可同时处理多个客户的请求
  - 客户和服务器都使用网络核心部分提供的服务

### 对等方式（P2P方式）
- **定义**：两台主机在通信时不区分服务请求方和服务提供方
- **特点**：
  - 平等的、对等连接通信
  - 对等连接中的每一台主机既是客户又是服务器
  - 可支持大量对等用户同时工作
  - 例如：BitTorrent、P2P文件共享
