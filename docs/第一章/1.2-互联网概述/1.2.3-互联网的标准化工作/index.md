# 1.2.3 互联网的标准化工作

## 组织架构
- **互联网协会（ISOC）**：负责互联网的总体协调
- **互联网体系结构研究委员会（IAB）**：管理IETF和IRTF
- **IETF（Internet Engineering Task Force）**：互联网工程任务组，负责互联网技术标准的制定（工程部）
- **IRTF（Internet Research Task Force）**：互联网研究任务组，负责长期研究项目（研究部）

## RFC文档
- **RFC（Request For Comments）**：请求评论文档
- **特点**：
  - 所有的互联网标准都是以RFC的形式发表的
  - 但并非所有RFC都是互联网标准（只有极少数RFC能成为正式标准）

## 制定标准的四个阶段
1. **互联网草案（Internet Draft）**
   - 有效期6个月，还不是RFC
2. **建议标准（Proposed Standard）**
   - 成为RFC
3. **草案标准（Draft Standard）**
   - 已取消
4. **互联网标准（Internet Standard）**
   - 记为STDxx
