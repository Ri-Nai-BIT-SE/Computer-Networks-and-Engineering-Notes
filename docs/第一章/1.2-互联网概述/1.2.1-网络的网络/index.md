# 1.2.1 网络的网络

## 核心概念

### 节点（Node）与链路（Link）
- **节点**：网络中的设备，可以是计算机、集线器、路由器、交换机等
- **链路**：连接节点的物理或逻辑通道
- **计算机网络（Network）**：由若干节点和连接这些节点的链路组成

### 网络与互连网（Internetwork）的区别
- **网络（Network）**：由节点和链路组成的单个通信系统
- **互连网（internetwork / internet）**：把多个网络通过路由器连接起来，构成覆盖范围更大的网络，即"网络的网络"
- **互联网（Internet）**：特指全球最大的、开放的、由众多网络互连而成的特定计算机网络
  - 采用 **TCP/IP 协议族**

### 用"云"表示网络
- 在拓扑图中，常用"云"的符号表示网络，表示网络内部的复杂性被抽象化
- **表示方法**：主机在"云"外，路由器在"云"内（核心）
