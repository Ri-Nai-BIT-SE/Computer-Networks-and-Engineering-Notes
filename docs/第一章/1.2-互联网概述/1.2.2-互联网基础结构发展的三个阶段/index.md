# 1.2.2 互联网基础结构发展的三个阶段

## 第一阶段（1969-1990）：从单个网络ARPANET向互联网发展
- **ARPANET**：最初是单个分组交换网
- **1983年**：TCP/IP协议成为ARPANET的标准协议，**互联网诞生**
- **特点**：实验性网络，主要用于军事和科研

## 第二阶段（1985-1993）：建成了三级结构的互联网（NSFNET）
- **结构**：
  1. **主干网**：连接各地区网络
  2. **地区网**：连接各校园网
  3. **校园网/企业网**：各大学和科研机构的局域网
- **特点**：层次化结构，由政府资助

## 第三阶段（1993-现在）：全球范围的多层次ISP结构
- **ISP（Internet Service Provider）**：互联网服务提供者（如中国电信、联通）
- **结构层次**：
  1. **主干ISP**：国家级或国际级的大型ISP
  2. **地区ISP**：区域性的ISP
  3. **本地ISP**：面向最终用户的ISP
- **IXP（Internet Exchange Point）**：互联网交换点
  - 允许两个网络（如两个ISP）直接相连并快速交换分组
  - 无需经过上层主干网
