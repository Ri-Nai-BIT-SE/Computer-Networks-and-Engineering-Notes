# 4.1.2 网络层的两个层面

网络层的功能可以划分为数据层面和控制层面，这是理解现代网络（特别是SDN）的基础。

## 1. 数据层面 (Data Plane)

- **核心动作**：转发 (Forwarding)。
- **定义**：路由器根据本路由器生成的**转发表**，把收到的分组从查找到的对应接口转发出去。
- **特点**：
  - 涉及**单个**路由器（局部操作）。
  - 通常采用**硬件**进行转发（速度快）。
  - 处理的是具体的数据分组。

## 2. 控制层面 (Control Plane)

- **核心动作**：路由选择 (Routing)。
- **定义**：根据路由选择协议所用的路由算法计算路由，创建出本路由器的**路由表**（进而生成转发表）。
- **特点**：
  - 涉及**许多**路由器协同动作（全局或分布式操作）。
  - 通常采用**软件**计算（速度相对较慢）。
  - 处理的是路由信息（为数据传送服务）。

## 3. 传统网络 vs SDN (软件定义网络)

- **传统网络**：控制层面和数据层面都**垂直集成**在每一个路由器中（每个路由器既管算路也管转发）。
- **SDN**：将**控制层面和数据层面分离**。
  - **远程控制器**（控制层面）：集中计算最佳路由，生成转发表下发给路由器。
  - **路由器/交换机**（数据层面）：只负责查找转发表进行转发，变得简单廉价。
