# 第四章 网络层

## 📋 考试注意事项（老师强调）

### 考试形式
- **试卷类型**：**B卷**
- **题型分布**：
  - **单项选择题**：10道（主要考查基本概念）
  - **综合题**：8道（这是拿分关键，涉及计算、分析和画图）

### 复习建议
- 老师会发布**复习范围**，明确哪些不考（如IPv6、组播、MPLS、SDN等已在考纲中划掉）
- 老师会提供一套**往年试题**，务必认真做，题型和难度参考价值极大
- **实验关联**：老师特别提到了**"静态路由实验"**。如果在实验中搞懂了**"下一跳地址怎么写"**（是写直连对端的接口IP），那么理论课的路由转发和转发表构建就没问题了

### ⚠️ 避坑指南（易错点）

1. **单位换算**：
   - IP首部长度字段单位是 **4字节**（数值5代表20字节）
   - 片偏移字段单位是 **8字节**
   - 总长度字段单位是 **1字节**

2. **子网划分**：注意"全0"和"全1"的主机号通常不可用，计算最大可分配地址时要减2

3. **转发表 vs 路由表**：综合题填表时，注意题目要求的是填"接口号"还是"下一跳IP地址"

4. **RIP的距离**：距离为16表示不可达。RIP交换的是整个路由表，而OSPF交换的是链路状态

5. **下一跳的逻辑**：老师反复强调**"以路由器的视角看世界"**
   - 如果是**直接交付**（在同一个局域网），不需要下一跳路由器
   - 如果是**间接交付**，下一跳必须是**直连链路**上的**对面接口的IP地址**，绝不能填更远处的路由器IP
